# Daily Collect

Daily Collect is an Android application designed to manage a **daily savings scheme (Pigmy)** with role-based access for administrators, agents, and customers. The app streamlines financial operations, ensures transparency, and provides secure access to savings data across devices.

---

## 📖 Overview
Daily Collect empowers financial institutions and agents to efficiently manage customer savings, withdrawals, and complaints. With Firebase integration, the app ensures secure data storage and real-time synchronization, making it accessible anytime.

---

## ✨ Features

### 👨‍💼 Administrator
- Add and manage customers and agents
- Adjust interest and commission rates
- Approve or reject withdrawal requests
- Oversee customer and agent records

### 🧑‍💻 Agent
- Add daily entries for customers
- Track personal earnings and commissions
- Manage customer lists
- Report issues directly through the app

### 👩‍👩‍👧 Customer
- View daily entries and savings history
- Apply for withdrawals
- Track balance and transactions
- Submit complaints seamlessly

## 🖼️ Screenshots

### Admin Interfaces
- **Login Screen** – Secure access for administrators  
- **Dashboard** – Quick navigation to customers, agents, approvals, complaints, and settings  
- **Withdrawal Requests** – Approve or delete customer withdrawal requests  
- **Interest Rate Settings** – Configure customer interest and agent commission rates  

### Agent Interfaces
- **Agent Dashboard** – Profile, earnings, add entry, customers, and issue reporting  
- **Add Entry** – Simple form to record daily savings  
- **Wallet/Earnings** – Track commissions and submit withdrawal requests  

### Customer Interfaces
- **Customer Dashboard** – Profile, balance, transaction history, withdrawal requests, complaints  
- **Profile Page** – View account details and personal information  
- **Transaction History** – Transparent record of savings and withdrawals

## ⚙️ Tech Stack
- **Frontend**: Android (Java/Kotlin)  
- **Backend**: Firebase Realtime Database  
- **Authentication**: Firebase Auth  
- **Storage**: Firebase Cloud Storage  

---

## 🚀 Installation
1. Clone the repository  
2. Open in Android Studio  
3. Configure Firebase project and add `google-services.json`  
4. Build and run on emulator or device
